# Contributing to AirVibe

Thank you for your interest in contributing to AirVibe.

AirVibe is an open environmental infrastructure project focused on education, citizen science, community monitoring and accessible environmental technology.

We welcome contributions from:
- developers
- students
- schools
- researchers
- makers
- educators
- environmental enthusiasts
- municipalities
- farmers and local communities

---

## Project Philosophy

AirVibe is built around:
- openness
- accessibility
- reproducibility
- education
- community collaboration
- low-cost environmental monitoring

The goal is not only to build sensors, but to create understandable environmental infrastructure that communities can learn from and expand.

---

## Ways to Contribute

You can contribute by helping with:

### Software
- Python scripts
- sensor integrations
- dashboards
- APIs
- automation
- deployment tools

### Documentation
- tutorials
- setup guides
- translations
- diagrams
- educational materials

### Hardware
- sensor testing
- Raspberry Pi deployments
- enclosure ideas
- agricultural monitoring
- field testing

### Education & Community
- workshops
- school projects
- citizen science initiatives
- environmental education
- community deployments

---

## Contribution Guidelines

Please try to:
- keep contributions clear and understandable
- document important changes
- write beginner-friendly explanations when possible
- respect the educational/community direction of the project

---

## Repository Structure

```text
docs/       → documentation
images/     → diagrams and screenshots
scripts/    → scripts and utilities
examples/   → example configurations and setups
diagrams/   → architecture and infrastructure diagrams
```

---

## Reporting Issues

If you find:
- bugs
- broken documentation
- sensor problems
- deployment issues
- visualization issues

please open an issue with:
- a short description
- steps to reproduce
- logs/screenshots if available

Please be respectful, constructive and supportive.

AirVibe is intended to be a welcoming project for both beginners and experienced contributors.
